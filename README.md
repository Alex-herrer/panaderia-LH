# Panadería L.H

Proyecto web de una panadería desarrollado como práctica personal mientras aprendo desarrollo web frontend.  
El proyecto incluye varias páginas de productos y un carrito de compras funcional.

## Descripción
Este proyecto simula una tienda web básica de una panadería, donde el usuario puede navegar entre diferentes categorías de productos, agregar artículos al carrito y ver el total a pagar.  
El carrito utiliza almacenamiento local del navegador para mantener la información aunque se recargue la página.

## Características
- Página principal con slider de imágenes
- Secciones de productos (panes, café, gaseosas y bebidas naturales)
- Carrito de compras funcional
- Contador de productos en el carrito
- Cálculo automático del total
- Página dedicada para visualizar el carrito
- Persistencia de datos con localStorage
- Diseño responsive básico

## Carrito de compras
- Agregar productos desde las diferentes páginas
- Aumentar y disminuir cantidades
- Eliminación automática cuando la cantidad llega a cero
- Total actualizado en tiempo real
- Datos guardados en el navegador

## Tecnologías utilizadas
- HTML5
- CSS3 (Flexbox y Grid)
- JavaScript (DOM y localStorage)
- Git
- GitHub
